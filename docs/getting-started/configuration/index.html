Configuration

ContextGraph reads its configuration from .contextgraph/config.json in your project directory. Run init to create a default file, or edit it manually.

Full config reference

.contextgraph/config.json
{
  "litellm": {
    "enabled": false,
    "baseUrl": "http://localhost:4000",
    "model": "gpt-4o",
    "rateLimitPerMinute": 10,
    "timeoutSeconds": 30
  },
  "includePatterns": ["**/*"],
  "excludePatterns": [
    "**/.git/**",
    "**/node_modules/**",
    "**/build/**",
    "**/.gradle/**",
    "**/.contextgraph/**",
    "**/target/**",
    "**/__pycache__/**"
  ],
  "maxFileSizeBytes": 10485760
}

Options

KeyDefaultDescription
litellm.enabledfalseEnable LLM-powered semantic extraction

litellm.timeoutSeconds30Per-call timeout in seconds
includePatterns["**/*"]Glob patterns for files to include
excludePatternsbuild dirs, .git…Glob patterns for files to skip
maxFileSizeBytes10485760 (10 MB)Files larger than this are skipped

Setting values via CLI

You can update config values without editing JSON directly:

Terminal
# Enable semantic extraction
./gradlew :modules:cli:run --args="config set litellm.enabled true"

# Point at a different model
./gradlew :modules:cli:run --args="config set litellm.model claude-opus-4-7"

# Change the LiteLLM endpoint
./gradlew :modules:cli:run --args="config set litellm.base-url http://localhost:4000"

# Increase max file size to 25 MB
./gradlew :modules:cli:run --args="config set max-file-size-mb 25"

Enabling semantic extraction

Semantic extraction sends file contents to an LLM to extract high-level concepts, decisions, and relationships. To use it:

  1. Start a LiteLLM proxy in front of your preferred model:
Terminal
pip install litellm
litellm --model claude-opus-4-7
  1. Enable it in your config:
Terminal
./gradlew :modules:cli:run --args="config set litellm.enabled true"
./gradlew :modules:cli:run --args="config set litellm.model claude-opus-4-7"
Note
Semantic extraction is rate-limited to rateLimitPerMinute calls. Set this low enough that you don't exceed your LLM provider's limits. The default of 10 calls/minute is conservative.

Filtering files

Use glob patterns to control which files get indexed. Patterns are matched against the full file path.

.contextgraph/config.json — Kotlin-only example
{
  "includePatterns": ["**/*.kt", "**/*.md", "**/*.sql"],
  "excludePatterns": ["**/build/**", "**/.gradle/**"]
}
