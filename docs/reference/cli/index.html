CLI Reference

All commands are run via Gradle:

Terminal
./gradlew :modules:cli:run --args="<command> [options]"
Working directory
All commands read .contextgraph/config.json and write to .contextgraph/graph.db relative to the current working directory, not the project being indexed.

init

Initialize .contextgraph/ in the current directory with a default config.json.

Terminal
./gradlew :modules:cli:run --args="init"

index

Index a directory into the knowledge graph. Skips files unchanged since the last run.

Terminal
./gradlew :modules:cli:run --args="index [path]"

# path defaults to "." (current directory)
./gradlew :modules:cli:run --args="index /path/to/project"

search

Full-text search across all graph nodes.

Terminal
./gradlew :modules:cli:run --args='search <query> [--limit N] [--min-confidence 0.5]'

./gradlew :modules:cli:run --args='search "authentication" --limit 10 --min-confidence 0.7'
FlagDefaultDescription
--limit20Maximum number of results
--min-confidence0.5Minimum node confidence score

node

Show a node's details and provenance by ID.

Terminal
./gradlew :modules:cli:run --args="node <nodeId>"

./gradlew :modules:cli:run --args="node io.example.AuthController"

expand

BFS neighborhood expansion from a node — returns all reachable nodes within the given depth.

Terminal
./gradlew :modules:cli:run --args="expand <nodeId> [--depth 2]"

path

Find the shortest path between two nodes.

Terminal
./gradlew :modules:cli:run --args="path <fromId> <toId>"

report

Generate GRAPH_REPORT.md and graph.html in the output directory.

Terminal
./gradlew :modules:cli:run --args="report [--output .]"

export

Export the full graph as a JSON snapshot.

Terminal
./gradlew :modules:cli:run --args="export [graph.json]"

serve-mcp

Start the MCP server over stdio. See MCP Server for full documentation.

Terminal
./gradlew :modules:cli:run --args="serve-mcp"

config set

Update a configuration value in .contextgraph/config.json.

Terminal
./gradlew :modules:cli:run --args="config set <key> <value>"
KeyExample value
litellm.enabledtrue
litellm.base-urlhttp://localhost:4000
litellm.modelclaude-opus-4-7
litellm.rate-limit10
max-file-size-mb25
